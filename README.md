# MusicNovice

[RU]
Данный проект направлен на создание сайта с рабочим названием "MusicNovice". Данный сайт предназначен для 
начинающих музыкантов, а также людей желающих научится играть на любом музыкальном инструменте.

[EN]
This project is aimed at creating website with the working title "MusicNovice". This site is intended for
young musicians, as well as people who want to learn to play any musical instrument.
